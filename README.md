## Installation ‚öôÔ∏è

1. Fork/Clone/Download this repo

    `git clone https://github.com/Datalux/Osintgram.git`

2. Navigate to the directory

    `cd Osintgram`

3. Start install.bat or use pip to download requirements

    `python main.py`
